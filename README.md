# Motion planning algorithm
Given the start point, this project aims to find the goal point in a 2D matrix . To do so, we will transform a BFS code in Greedy code.

**General notation**
* 0: free space
* 1: ocupued (wall/obstacle)
* 2: visited point
* 3: start point
* 4: goal point
